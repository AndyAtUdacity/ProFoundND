New Terms

HTML
HTML stands for Hypertext Markup Language. When we write HTML we are creating a structured document which a web browser (like Chrome of Firefox) can understand and display.

HTML is not a full programming language: you can't use it to do calculations or build software.
CSS
CSS stands for Cascading Style Sheet. While the HTML gives the overall structure of web pages, without any styling these pages would be pretty boring.

CSS addresses this by specifying the visual properties (position, size, color, etc...) of different HTML elements.
Elements
Tags
Attributes
DOM
Short for "Document Object Model." This refers to the "structure" of HTML documents.
